0x1. 미러링이 제대로 안될 때

  • 앱을 최신 버전으로 업데이트 합니다. 상당히 많은 개선사항이 추가되었기 때문에 반드시 업데이트 부탁드립니다.
    최소 Pintot 1.3.2 버전 이상 사용이 필수 입니다.

  • 앱의 오른쪽 위에 있는 X 버튼을 눌러서 완전히 종료합니다.

  • 완전 종료 후에도 효과가 없다면 앱을 삭제 후 재설치합니다.

  • 터치가 안되는 경우 안드로이드에서 일시적인 접근성 문제가 있을 수 있습니다.
    이 경우 세팅의 접근성 메뉴를 껐다 다시 켜시거나, 스마트폰을 재부팅 하는 것이 효과적입니다.

  • 주행 중에만 안되는 경우 대부분은 차량 wifi 설정에 \"주행 중에 연결 상태 유지\" 옵션이 꺼져있어서 입니다.
    차량에서 hotspot 접속 후 반드시 해당 옵션 체크 부탁드립니다.

0x0. Basic Interface

  • (Expand) Have you checked the Pintot operation? Make sure the light is on for button and self-test link is working properly.
  • When you connect to 3.3.3.3:4567 from a Tesla vehicle browser, you will be automatically redirected to 3.3.3.3:4567/client.html.
  • The basic interface buttons are as follows:

Refresh

Use this button to refresh the screen if the mirroring screen is not displayed.

Disconnect

Mirrorring will be temporarily quit. It will be valid until next refresh.

Zoom In/Out

This button zooms in or out to the largest screen that can display the mirroring screen.

Home Button (For mirroring phone)

Send the home button action to the mirroring phone.

Back Button (For mirroring phone)

Send the back button action to the mirroring phone.

Movie Theater Mode (only while parked)

You will be redirected to a page like a simplified OTT hub that allows you to use each OTT service in fullscreen mode.

No further reconfirmation is required until it is completely turned off.



0x1. When mirroring doesn't work properly

  • Update the app to the latest version. Numerous improvements have been added, so updating is highly recommended.
    Pintot Version 1.3.2 or higher is required at a minimum.

  • press the X button in the right-top corner of the app to completely exit.

  • If the issue persists even after a complete exit, uninstall the app and reinstall it.

  • If touch input isn't working, it might be due to a temporary accessibility issue on Android.
    In this case, turning the Accessibility settings off and then on again, or restarting your smartphone may resolve the issue.

  • If it doesn't work only while driving, this is usually because the option "Maintain connection while driving" is turned off in the vehicle's Wi-Fi settings.
    After connecting to the vehicle's hotspot, please make sure this option is checked.



0x2. When screen transmission is slow

    ◈ General Solutions

  • > Optimization is nearly complete, making further speed improvements difficult.
    > In practice, devices with performance equivalent to or better than the S25 show significant improvement in speed and reduced lagging issues.

  • Exit the screen using the browser's back button, then reconnect.

  • Turn off vehicle Wi-Fi, then turn it back on and reconnect to Pintot.

  • Close Pintot and reopen it, or restart your smartphone.

  • Exit the screen using the browser's back button, then reconnect to the client.

    • ◈ Last Resort Solutions

  • > The Tesla browser applies resource management policies unfavorable to our app, so we recommend using the above methods to mitigate this issue.
    > However, if the app still slows down, it's likely due to insufficient resources, which may rarely occur on the latest software version.
    > In such cases, use the last resort solutions below.

  • Press and hold both scroll buttons on the steering wheel to restart the touchscreen.

  • The speed immediately after rebooting represents the maximum mirroring performance achievable on the current software version.
